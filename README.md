
**FastAPI CryptoMarket**

A FastAPI application designed to provide real-time cryptocurrency market data and related functionalities.

Features
• Fetch real-time cryptocurrency prices
• Retrieve historical market data
• User authentication and profile management
• Portfolio tracking

Installation
1. Clone the repository:
git clone https://github.com/AmirSomo/fastapi_cryptomarket.git
cd fastapi_cryptomarket

2. Create and activate a virtual environment:
python3 -m venv venv
source venv/bin/activate

3. Install dependencies:
pip install -r requirements.txt


Usage
1. Start the FastAPI server:
uvicorn main:app --reload

2. Access the API documentation:
Navigate to http://127.0.0.1:8000/docs in your browser to explore the available endpoints and interact with the API.
